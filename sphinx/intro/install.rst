
Install
-------

Sty exists as a `pypi package <https://pypi.org/project/sty/>`__.

Install with `pip <https://pip.pypa.io/en/stable/>`__:

::

   pip install sty

or with `pipenv <https://pipenv.pypa.io/en/latest/>`__

::

   pipenv install sty


or with `poetry <https://python-poetry.org/>`__

::

   poetry add sty
